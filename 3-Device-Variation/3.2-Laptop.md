## 3.2.1 - Automated setup script

Run the laptop setup script. This installs laptop-specific packages, enables power profiles, and applies laptop kernel parameters.

```
chmod +x ~/Obsidian/AMD-Linux-Setup/Scripts/laptop_setup.sh && ~/Obsidian/AMD-Linux-Setup/Scripts/laptop_setup.sh
```

## 3.2.2 - Power management configuration

The `power-profiles-daemon` service was enabled by the script. You should now configure your specific preferences in KDE.

1. Open **System Settings**.
    
2. Navigate to **Energy Saving**.
    
3. Configure **AC Power**, **Battery**, and **Low Battery** settings to your preference.

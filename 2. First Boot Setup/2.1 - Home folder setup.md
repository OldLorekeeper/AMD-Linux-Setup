## 2.1.2 - Create @games subvolume

First, identify your main BTRFS partition device name (e.g., `/dev/vda2`) and its UUID.

```
lsblk -f
```

Mount `/dev/[btrfs-root]` to create the new Games subvolume:

```
# Replace [btrfs-root] with your device name
sudo mount /dev/[btrfs-root] /mnt -o subvolid=5

# Create the top-level subvolume
sudo btrfs subvolume create /mnt/@games

# Unmount the temporary root
sudo umount /mnt
```

Edit fstab to correctly automount:

```
kate /etc/fstab
```

Add the following entry, replacing `user` and `uuid_string`placeholders:

```
# /home/[user]/Games
UUID=[uuid_string]	/home/[user]/Games	btrfs     	rw,noatime,compress=zstd:3,ssd,discard=async,space_cache=v2,subvol=/@games	0 0
```

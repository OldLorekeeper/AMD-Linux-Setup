## 2.5.5 - Add pacman hooks

Create executable script:

```
sudo tee /usr/local/bin/update-init-grub > /dev/null <<EOF && sudo chmod +x /usr/local/bin/update-init-grub
#!/bin/bash
# Replace the below commands with the actual commands that your aliases refer to.
sudo mkinitcpio -P
sudo grub-mkconfig -o /boot/grub/grub.cfg
EOF
```

Create hook:

```
sudo tee /etc/pacman.d/hooks/99-update-init-grub.hook > /dev/null <<EOF
[Trigger]
Operation = Install
Operation = Upgrade
Type = Package
Target = *

[Action]
Description = Updating initramfs and GRUB configurations
When = PostTransaction
Exec = /usr/local/bin/update-init-grub
EOF
```

## 2.5.13 - Delay Steam autostart and launch in system tray only

Create executable script:

```
echo -e '#!/bin/bash\nsleep 15\n/usr/bin/steam -silent "$@"' > ~/Make/steam-delay.sh && chmod +x ~/Make/steam-delay.sh
```

Open System Settings and add Steam to autostart:

```
Autostart >> Add new >> Application >> Games >> Steam >> OK
```

Edit Steam Autostart entry to launch steam-delay.sh:

```
Click 'properties' menu >> Application
> Program: ~/Make/steam-delay.sh
> Arguments: %U
```

<sup>Note: From experience, if Steam starts too early then the browser elements won't connect to the network properly and will return error codes</sup>

## 2.5.14 – Configure Reflector Service and Timer

**Purpose:**  

Automatically update the Arch Linux mirrorlist immediately after boot and then every 6 hours thereafter using _reflector_.

<br/>

### Service Unit

`kate /etc/systemd/system/reflector-update.service`

Add the following:

`[Unit] Description=Update Arch mirrorlist using Reflector Wants=network-online.target After=network-online.target  [Service] Type=oneshot ExecStart=/usr/bin/reflector --ipv4 -c GB -p https --download-timeout 5 --age 8 --latest 30 --fastest 10 --sort rate --save /etc/pacman.d/mirrorlist Nice=19 IOSchedulingClass=idle  [Install] WantedBy=multi-user.target`

<br/>

### Timer Unit

`kate /etc/systemd/system/reflector-update.timer`

Add the following:

`[Unit] Description=Run reflector-update 6h after boot, then every 6h  [Timer] # First run occurs 6 hours after boot OnBootSec=6h # Subsequent runs every 6 hours after the last run OnUnitActiveSec=6h Persistent=true  [Install] WantedBy=timers.target`

<br/>

### Enable and Verify

Enable the service for immediate post-boot execution:

`sudo systemctl enable reflector-update.service`

Enable the timer for subsequent scheduled runs:

`sudo systemctl enable --now reflector-update.timer`

Ensure the system waits for an active network connection:

`sudo systemctl enable --now NetworkManager-wait-online.service`

Check timer scheduling:

`systemctl list-timers | grep reflector-update`

<sup>Note:</sup>

- The service runs once automatically at boot when the network is ready.
    
- The timer triggers 6 hours after boot, and then every 6 hours thereafter.
    
- Adjust `--download-timeout` or `--age` if you prefer more aggressive or lenient mirror filtering.

This script automates the core system setup by:

- Optimising `/etc/makepkg.conf` for native compilation and faster package building.
    
- Updating the mirror list with the fastest UK servers.
    
- Installing the `yay` AUR helper.
    
- Installing all common packages from `core_pkg.txt`.
    
- Enabling core system services.
    

Run the following command:

```
chmod +x ~/Obsidian/AMD-Linux-Setup/Scripts/core_setup.sh && ~/Obsidian/AMD-Linux-Setup/Scripts/core_setup.sh
```

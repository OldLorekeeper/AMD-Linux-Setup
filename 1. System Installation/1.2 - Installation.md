## 1.2.1 - Connect to WiFi

```
>>> iwctl
>>> station wlan0 connect '[ssid]'
```

## 1.2.2 - Run setup script

```
curl -sL "https://raw.githubusercontent.com/OldLorekeeper/AMD-Linux-Setup/main/Scripts/systeminstall.sh" | bash
```

## 1.2.3 - Make system specific amends

```
>>> Disk configuration > Partitioning > Best effort > [disk] > btrfs > Default? Yes > Compression? Yes
>>> Btrfs snapshots > Timeshift
>>> Hostname > [device_name]
>>> Root password > [password] > Confirm
>>> User account > Add user/password > Superuser? Yes > Confirm and exit
```

<sub> Note: replace square brackets </sub>

<br/>
 
## 1.2.4 - Begin installation

Select `Install` at bottom of archinstall options

<br/>

## 1.2.5 - Assign user to groups

Chroot into system and run the following commands:

```
groupadd --gid 102 polkit && systemd-sysusers
```

<sub> Comment: This is a just-in-case fix from prior experience </sub>

Add user to sudo and input groups:

```
usermod -aG wheel,input [username]
```

<sub> Note: replace square brackets </sub>

<br/>
 
## 1.2.6 - Reboot

Press Ctrl+D to exit chroot and run:

```
reboot
```
